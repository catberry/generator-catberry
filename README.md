## Installation

```bash
npm install yo -g
npm install generator-catberry -g
```

Then generate your new project:

```bash
mkdir my-new-catberry-app
cd my-new-catberry-app
yo catberry [<appTemplate>]
```

See catberry/catberry-cli for available app templates.
